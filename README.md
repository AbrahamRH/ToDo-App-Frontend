# ToDo-App-Frontend
Frontend application for the Todo App list project.

## Install npm

```
npm install
```

## run tests

```
npm run test
```

## Launch the App

To use the app you need to launch the [ToDo-App-Backend](https://github.com/AbrahamRH/ToDo-App-Backend.git) first.

Then you can launch this project with:

```
npm run start
```
